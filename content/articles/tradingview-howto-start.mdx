## TradingViewとは

TradingViewは世界で最も使われているチャートプラットフォームのひとつです。ブラウザ上で動作し、為替・株・仮想通貨など幅広い銘柄のチャートを無料で確認できます。

## 無料プランでできること

| 機能 | 無料（Basic） |
|------|-------------|
| チャート表示 | ◎ |
| インジケーター追加 | 最大3つ同時 |
| アラート設定 | 最大1件 |
| Pine Scriptエディタ | ◎ |
| マルチチャート | 1画面のみ |

## 始め方

1. tradingview.com にアクセス
2. 右上「登録」をクリック（メールまたはGoogleアカウントで登録）
3. チャート画面を開く

## 基本画面の説明

**上部ツールバー**
- 銘柄検索ボックス — 「USDJPY」などを入力
- 時間足セレクター — 1m/5m/15m/1h/4h/1D など
- インジケーターボタン — クリックして追加
- アラートボタン（時計アイコン）

**左側ツールバー**
- トレンドライン・水平線・フィボナッチなどの描画ツール

**右側パネル**
- ウォッチリスト（お気に入り銘柄一覧）

## インジケーターの追加方法

1. 上部「インジケーター」ボタンをクリック
2. 検索欄に名前を入力（例：「RSI」）
3. 一覧から選んでクリック
4. チャートに自動で追加される

## Pine Scriptエディタの開き方

1. 画面下部「Pineエディタ」タブをクリック
2. コードを入力
3. 「チャートに追加」ボタンをクリック

## チャートのレイアウト保存

チャートのレイアウト（インジケーター配置・描画など）はクラウドに自動保存されます。上部の「保存」ボタンで任意のタイミングで保存もできます。

## おすすめの最初の設定

```
時間足：1時間足（1H）
銘柄：USDJPY（ドル円）
インジケーター：
  - EMA 20（青）
  - EMA 50（オレンジ）
  - RSI 14（サブチャート）
```

この3つを追加するだけで、相場のトレンドと過熱感をひと目で確認できます。

## まとめ

TradingViewはブラウザだけで使え、インストール不要です。まずは無料プランで基本操作に慣れ、Pine Scriptの学習に進んでいきましょう。
