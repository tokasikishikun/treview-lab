## RCIとは

RCI（Rank Correlation Index・順位相関指数）は、時間の順位と価格の順位の相関を−100〜+100で表す日本独自のオシレーターです。

## 計算方法

```
d = 各バーの「時間順位」と「価格順位」の差の二乗和
RCI = (1 - 6d / (n(n²-1))) × 100
```

- +100に近い → 時間の経過とともに価格が上昇している（上昇トレンド強い）
- −100に近い → 時間の経過とともに価格が下落している（下降トレンド強い）
- 0付近 → トレンドなし

## Pine Scriptで実装

```pine
//@version=5
indicator("RCI", overlay=false)

rci(src, len) =>
    d = 0.0
    for i = 0 to len - 1
        // 時間順位（新しいほど順位が高い）
        timeRank = len - i
        // 価格順位
        priceRank = 1
        for j = 0 to len - 1
            if src[j] < src[i]
                priceRank += 1
        d += math.pow(timeRank - priceRank, 2)
    1.0 - 6.0 * d / (len * (math.pow(len, 2) - 1.0))

length = input.int(9, "RCI期間")
rciVal = rci(close, length) * 100

hline(80,  "買われすぎ", color=color.red,   linestyle=hline.style_dashed)
hline(-80, "売られすぎ", color=color.green, linestyle=hline.style_dashed)
hline(0,   "中立",       color=color.gray,  linestyle=hline.style_dotted)

rciColor = rciVal >= 0 ? color.blue : color.red
plot(rciVal, title="RCI", color=rciColor, linewidth=2)
```

## 3本RCI（短期・中期・長期）

3本のRCIを組み合わせることで精度が上がります。

```pine
//@version=5
indicator("3本RCI", overlay=false)

rci(src, len) =>
    d = 0.0
    for i = 0 to len - 1
        timeRank  = len - i
        priceRank = 1
        for j = 0 to len - 1
            if src[j] < src[i]
                priceRank += 1
        d += math.pow(timeRank - priceRank, 2)
    (1.0 - 6.0 * d / (len * (math.pow(len, 2) - 1.0))) * 100

rciS = rci(close, 9)   // 短期
rciM = rci(close, 26)  // 中期
rciL = rci(close, 52)  // 長期

hline(80,  color=color.red,   linestyle=hline.style_dashed)
hline(-80, color=color.green, linestyle=hline.style_dashed)
hline(0,   color=color.gray,  linestyle=hline.style_dotted)

plot(rciS, title="短期(9)",  color=color.blue,   linewidth=1)
plot(rciM, title="中期(26)", color=color.orange, linewidth=2)
plot(rciL, title="長期(52)", color=color.red,    linewidth=2)
```

## 3本RCIのエントリー条件

**買いエントリー**
- 長期RCIが−80以下から上昇
- 中期RCIも−80以下
- 短期RCIが−80から上昇反転

**売りエントリー**
- 長期RCIが+80以上から下落
- 中期RCIも+80以上
- 短期RCIが+80から下落反転

## まとめ

RCIはRSIに比べて反応が遅めですが、**3本を組み合わせると上位足のトレンド方向に乗る精度の高いエントリーポイント**を見つけやすくなります。日本のFXトレーダーに特に愛用されているインジケーターです。
