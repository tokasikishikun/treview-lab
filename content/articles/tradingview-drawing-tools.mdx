## 描画ツールの場所

チャート左側のツールバーに描画ツールが並んでいます。各アイコンにホバーするとツール名が表示されます。

## よく使う描画ツール

### トレンドライン
高値同士・安値同士を結ぶ斜めのライン。

**使い方**
1. ツールバーから「トレンドライン」を選択
2. 始点をクリック → 終点をクリック
3. `Shift`を押しながらドラッグで水平・垂直・45度に固定

### 水平線
サポート・レジスタンスに使う横線。

**ショートカット** — `Alt + H`

### 水平レイ（片方向に延長）
片方向のみ延長される水平線。サポレジに便利。

### フィボナッチリトレースメント
押し目・戻りの目安を自動計算するツール。

**使い方**
1. 「フィボナッチリトレースメント」を選択
2. 上昇波の場合：安値から高値にドラッグ
3. 0.382・0.5・0.618のラインが自動で引かれる

**主なレベルの意味**

| レベル | 意味 |
|--------|------|
| 0.236 | 浅い押し目 |
| 0.382 | 軽い押し目（よく反発） |
| 0.500 | 中間値 |
| 0.618 | 深い押し目（黄金比・強い支持） |
| 0.786 | 非常に深い押し目 |

### 平行チャンネル
トレンドラインと平行なチャンネルを描くツール。

**使い方**
1. 「平行チャンネル」を選択
2. 2点でトレンドラインを引く
3. 3点目でチャンネルの幅を設定

## ショートカットキー一覧

| ショートカット | 操作 |
|--------------|------|
| `Alt + T` | トレンドライン |
| `Alt + H` | 水平線 |
| `Alt + F` | フィボナッチ |
| `Alt + V` | 垂直線 |
| `Ctrl + Z` | 直前の描画を取り消し |
| `Backspace` | 選択した描画を削除 |
| `Ctrl + A` | 全描画を選択 |

## 描画のロック機能

誤って動かさないようにロックできます。

1. 描画を右クリック
2. 「ロック」をクリック

全描画をロックする場合：チャート上部「描画ツール」→「全ての描画をロック」

## テンプレートへの保存

よく使う描画スタイル（色・太さ・スタイル）はテンプレートとして保存できます。

1. 描画を右クリック → 「スタイルを保存」
2. テンプレート名を入力
3. 次回から右クリック → 「スタイルを適用」で再利用

## まとめ

TradingViewの描画ツールを使いこなすことで、チャート分析の速度が大幅に上がります。まずはトレンドライン・水平線・フィボナッチの3つをマスターすることを目標にしましょう。
